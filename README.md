# Projeto do DS2 - ETE



Este é um projeto [Next.js](https://nextjs.org) inicializado com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).



## 🚀 Instalação e Configuração



Quando você faz o clone deste repositório, a pasta `node_modules` não é incluída no download. Isso é um comportamento padrão (configurado no `.gitignore`) para evitar que arquivos pesados e gerados automaticamente sejam enviados para o controle de versão.



Para baixar e instalar todas as dependências necessárias para rodar o projeto, abra o terminal na pasta raiz do projeto e execute o comando abaixo:



```bash

npm install

```



## 💻 Executando o Servidor de Desenvolvimento



Após a instalação das dependências, você pode iniciar o servidor de desenvolvimento:



```bash

npm run dev



Abra http://localhost:3000 no seu navegador para ver o resultado.
